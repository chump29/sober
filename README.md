## Development stuff

To manually lint files:

```bash
pnpm run lint
```

To manually run tests:

```bash
pnpm run test:dev # hot reloads
# or
pnpm run test # not interactive
```

To manually build files:

```bash
pnpm run build:dev #DEV
# or
pnpm run build #PROD
```

## Docker Stuff

To build:

```bash
./build.sh
```
